# Tura 🚗

**La plataforma de alquiler de autos entre particulares para Latinoamérica**

Airbnb de autos - Conecta propietarios con viajeros en toda América Latina.

- 🇦🇷 Argentina
- 🇧🇷 Brasil
- 🇨🇱 Chile
- 🇨🇴 Colombia

- 🇪🇨 Ecuador
- 🇲🇽 México
- 🇵🇦 Panamá
- 🇵🇪 Perú
- 🇺🇾 Uruguay
- 🇻🇪 Venezuela
- 🇩🇴 República Dominicana

## 🚀 Características

- ✅ **Búsqueda de autos** - Encuentra el auto perfecto por ubicación y fechas
- ✅ **Perfiles de anfitriones** - Propietarios verificados con calificaciones
- ✅ **Seguro incluido** - Todos los autos tienen cobertura completa
- ✅ **Soporte 24/7** - Equipo de soporte en español y portugués
- ✅ **Pagos seguros** - Integración con métodos de pago locales
- ✅ **Diseño responsive** - Funciona en móvil, tablet y desktop

## 🛠️ Tech Stack

- **Frontend:** React 18 + Vite
- **Estilos:** Tailwind CSS
- **Animaciones:** Framer Motion
- **Iconos:** Lucide React
- **Mapas:** (próximamente) Google Maps API
- **Pagos:** (próximamente) Stripe/MercadoPago

## 📦 Instalación

```bash
# Clonar repositorio
git clone https://github.com/xmrtdao/tura-frontend.git
cd tura-frontend

# Instalar dependencias
npm install

# Iniciar servidor de desarrollo
npm run dev

# Construir para producción
npm run build
```
