# Contributing to XMRT DAO Projects

Thank you for your interest in contributing! This document provides guidelines for participating in our projects.

## How to Contribute

1. **Fork the repository** and create your branch from `main`.
2. **Make your changes** — add tests if applicable.
3. **Ensure the test suite passes** (when available).
4. **Update documentation** if your changes affect usage.
5. **Submit a pull request** with a clear description.

## Code of Conduct

All contributors are expected to adhere to our code of conduct: be respectful, constructive, and inclusive.

## Reporting Issues

- Use GitHub Issues to report bugs or request features.
- Provide clear reproduction steps for bug reports.
- Search existing issues before creating new ones.

## Security

For security concerns, please review our SECURITY.md or contact the maintainers directly rather than opening public issues.

## Questions?

Reach out via GitHub Discussions or the project documentation.
